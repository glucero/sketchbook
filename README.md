# sketchbook
### a collection of images and animations

requirements

    clojure   (http://clojure.org)
    leiningen (https://github.com/technomancy/leiningen)
    quil      (https://github.com/quil/quil)

creating a sketch

    ./sketch 'name'

launching a sketch

    cd ./sketch_name
    lein deps
    lein run
